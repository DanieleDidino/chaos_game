## Chaos game

**chaos_game.py** creates a fractal using the 
[chaos game](https://en.wikipedia.org/wiki/Chaos_game)
method. The shape of the fractal is based on the number of vertices.
It is based on [this](https://youtu.be/kbKtFN71Lfs) Numberphile video.
